# Лендинг «Узаконить перепланировку» (React)

## Сборка и запуск
1. Установите Node.js.
2. В папке проекта выполните: `npm install`
3. Для разработки: `npm start`
4. Для продакшен-сборки: `npm run build`

## Загрузка на Beget
- Соберите проект: `npm run build`
- Через файловый менеджер Beget загрузите **всё содержимое** папки `build` в `public_html` вашего сайта.
- Готово! Лендинг работает.

## Опросник
Пользователь отвечает на 3 вопроса, после чего получает сообщение с результатом. Можно легко подключить отправку данных на email через бэкенд (например, FormSubmit).
